## 算法符号说明


### 矩阵表示符号
  表示矩阵某一行或列的符号——“:”。
 
  如果 A 是 n 阶方阵，则
 * A (k,:) 表示矩阵的第 k 行，即：

    A(k,:) = (a[k,1], a[k,2], …, a[k,n])

 * A(:,k)表示矩阵 A 的第 k 列，即：

    A(:,k) = (a[1,k], a[2,k], …, a[n,k])^T

 * 更一般的，如果整数p, q和 k 满足 1 ≤ p ≤ q ≤ n 和 1 ≤ k ≤ n，则：

    A(k,p:q) = (a[k,p], a[k,p+1], …, a[k,q])

    A(p:q,k) = (a[p,k[, a[p+1,k], …, a[q,k])^T


### 赋值符号
 * 符号“：=” 与 “←” 均表示赋值或传输。
 * “%”后面的内容表示对表达式的说明。
